# NGO Program Monitoring Dashboard

This is a simple web-based dashboard designed to demonstrate how NGOs can visualize
program activities, coverage, and operational data.

## Features
- Summary cards for key indicators
- Program activity table
- Responsive and lightweight design
- Built with HTML, CSS, and JavaScript

## Use Case
Suitable for internal program monitoring, MEL reporting, and operational oversight.
